Getting Started

Dynamic forms is an Angular JS module that allows you to dynamic generate a HTML form based off of standard JSON schema complete with support for the following:
  • Form Validation
  • Extensible and customizable template driven design
  • Default templates leverage bootstrap, but can be fully customized

Installation

Install with bower

$ bower install angular-dynamic-forms

Add script reference to your app

<script type="text/javascript" src="bower_components/dist/dynamic-forms.js"></script>

Add Angular Reference

angular.module('myApp', ['dynamic-forms']);

Add dynamic-form directive to your view

<dynamic-form schema="formSchema" ng-model="formModel"></dynamic-form>

Basic Example

JavaScript
vm.formSchema = {
    type: 'object',
    title: 'Basic Example',
    description: 'Example description for angular dynamic form',
    properties: {
        firstName: { type: 'string', title: 'First Name', description: 'Your first name', required: true },
        lastName: { type: 'string', title: 'Last Name', description: 'Your last name', required: true },
        email: { type: 'string', format: 'email', title: 'E-mail', description: 'Your email address', required: true }
    }
};
vm.formModel = {
    firstName: 'John',
    lastName: 'Doe',
    email: 'john.doe@gmail.com'
};
HTML
<dynamic-form schema="home.formSchema" ng-model="home.formModel">
    <button type="button" class="btn btn-success">Submit</button>
    <button type="button" class="btn btn-danger">Cancel</button>
</dynamic-form>
